# Simulating the Casimir Force for Nontrivial Geometries
Casimir force simulation in nontrivial geometries using worldline numerics and Monte Carlo methods, with visualization of vacuum fluctuation contributions and breakdown of the proximity force approximation (PFA).

This project simulates the Casimir Effect in nontrivial geometries using a Monte Carlo approach known as worldline numerics.

Instead of solving electromagnetic field equations directly, the Casimir interaction is approximated by sampling large ensembles of random closed paths (“worldlines”) and determining how they interact with bounding surfaces.

The goal is to study how geometry influences vacuum forces, and to investigate the limitations of common approximations such as the Proximity Force Approximation.
